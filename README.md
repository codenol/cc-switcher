# cc-switcher

**Работаете за одним маком в несколько человек? Надоело каждую смену выходить из Claude и логиниться заново через почту и письма со ссылками?**

Есть решение.

cc-switcher живёт в верхней панели мака — там же, где часы и Wi-Fi — и переключает Claude Desktop между аккаунтами **в один клик**. Без выхода вручную, без ввода почты, без поиска письма со ссылкой. Нажал — и через пару секунд Claude уже открыт под нужным человеком.

---

## Зачем это нужно

Один компьютер. Несколько человек по сменам. У каждого свой аккаунт Claude.

Раньше смена начиналась так: выйти из чужого аккаунта → нажать Get Started → ввести почту → открыть почту → найти письмо от Claude → перейти по ссылке → вернуться в Claude. Каждый раз. Каждая смена.

Теперь — открыл меню в панели, выбрал своё имя, всё.

## Что умеет

- **Время до сброса прямо в панели.** Рядом с именем активного аккаунта видно, во сколько сбрасывается его лимит — не заходя никуда.
- **Переключение в один клик.** Меню → «Переключиться» → выбрать человека. Claude сам перезапустится под выбранным аккаунтом.
- **Все аккаунты под рукой.** В меню — список всех, у каждого видно время до сброса.
- **Настройки с хранением логинов.** Логин (почта), пароль, пароль от почты и ссылка на почтовый сервис — всё в одном окне, аккуратно и под рукой.

## Как выглядит работа

1. В панели сверху всегда видно активного и его сброс: `Вася · 20:30`.
2. Клик по иконке → меню:
   - **Переключиться** ▸ — список аккаунтов с временем сброса, выбрать нужный
   - **Настройки…** — управление аккаунтами
   - **Выйти**
3. Выбрали аккаунт → приложение спросит, во сколько сбрасывается лимит **уходящей** сессии, переключит Claude, а затем спросит время сброса **новой**. Пара секунд — готово.

> Время сброса вводится вручную: живой остаток claude.ai закрыт защитой Cloudflare, поэтому часы:минуты ресета задаёте вы при переключении (и можете обновить, кликнув по уже активному аккаунту).

## Безопасность

Пароли хранятся в системной связке ключей macOS (Keychain), а не в открытом файле. Данные не покидают ваш компьютер.

## Требования

- macOS
- Установленный Claude Desktop

## Первый запуск

При первом переключении macOS один раз спросит разрешение на доступ к данным Claude — нажмите **«Всегда разрешать» (Always Allow)**. Больше спрашивать не будет.

Чтобы добавить человека: пусть он один раз войдёт в Claude Desktop как обычно, затем в настройках нажмите **«Захватить текущий аккаунт»** — и дальше он переключается одной кнопкой.

## Сборка из исходников

Нужны Node.js, Rust (стабильный) и Xcode Command Line Tools.

```bash
npm install
npm run tauri build
```

Готовый `.app` появится в
`src-tauri/target/release/bundle/macos/cc-switcher.app`,
а `.dmg` — в `src-tauri/target/release/bundle/dmg/`.

### Подпись и Gatekeeper

Сборка по умолчанию подписывается **ad-hoc** (без сертификата Apple Developer) — этого достаточно для личного использования на своей машине. Но при первом запуске Gatekeeper заблокирует «приложение от неустановленного разработчика». Обойти разово:

- **Через Finder:** правый клик по `cc-switcher.app` → **Открыть** → в диалоге ещё раз **Открыть**. Дальше запускается обычным двойным кликом.
- **Или из терминала** (снять карантин):
  ```bash
  xattr -dr com.apple.quarantine /Applications/cc-switcher.app
  ```

Если есть сертификат **Developer ID**, можно подписать им и нотаризовать — тогда диалога Gatekeeper не будет. Задаётся через переменную окружения `APPLE_SIGNING_IDENTITY` перед `npm run tauri build`.

### Автозапуск

Чтобы приложение поднималось при входе в систему: **Системные настройки → Основные → Объекты входа → +** и добавить `cc-switcher.app`.
